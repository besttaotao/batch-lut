# 批量色彩还原工具 (Batch LUT Restoration Tool)

一款专为 **DJI Osmo Action 5/4**、**Pocket 3** 等设备开发的桌面端批量色彩还原工具。支持 D-Log M 素材一键转标准色彩（Rec.709），并具备自适应原片规格的无损导出能力。

---

## 📥 下载即用 (Direct Download)

对于普通用户，**无需配置任何编程环境**，直接下载即可使用：

1.  前往 [**GitHub Releases**](https://github.com/besttaotao/batch-lut/releases) 页面。
2.  下载最新的 **`batch-lut-app-1.0.0-setup.exe`** 安装包。
3.  双击运行安装（若浏览器提示风险，请选择“保留”或“仍要运行”）。
4.  **本正式版已内置 FFmpeg**，安装完成后即可开始转换。

---

## 🌟 核心特性

- **规格自适应**：自动识别原片分辨率、帧率及码率，确保还原后的视频规格 1:1 对齐。
- **全平台硬件加速**：支持 NVIDIA (NVENC) 和 AMD (AMF) 显卡加速，以及通用 CPU 模式。
- **无损画质**：默认采用 H.265 (HEVC) 编码，支持 100Mbps+ 比特率匹配，几乎无损还原。
- **开箱即用**：Release 版本已内置所有引擎，用户无需额外安装任何环境。

---

## 🚀 操作手册

1.  **添加视频**：拖拽 MP4 或点击中心区域选择文件。
2.  **选择 LUT**：选取对应的 `.cube` 还原文件。
3.  **设置目录**：指定导出位置（支持防覆盖重命名）。
4.  **选择模式**：根据显卡型号选择对应的加速模式。
5.  **一键导出**：点击开始，支持随时中止并自动清理缓存。

## 🛠️ 开发者指南 (Developer Guide)

如果你希望在本地克隆并运行本项目进行二次开发，请阅读以下说明。

### 1. 环境要求

- **Node.js**: 建议 v18.0 或更高版本。
- **Git**: 用于版本管理。

### 2. 核心依赖

- **内置引擎**：为了方便开发，本项目已在 `resources/bin/` 目录下内置了 Windows 版 `ffmpeg.exe` 和 `ffprobe.exe`。克隆仓库后无需额外配置。

### 3. 项目运行

```bash
npm install     # 安装依赖
npm run dev     # 启动开发预览
npm run build:win  # 打包生成 EXE
```
